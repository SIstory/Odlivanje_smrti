  • DCTERMS - razširjeni metapodatkovni standard DCMI Metadata Terms

  • EAD - Encoded Archival Description
  • EN15907 standard - standard za metapodatkovni zapis kinematografskih del
  • ERIC - European Research Infrastructure Consortium (Evropski raziskovalni infrastrukturni konzorcij)
  • GAMS - Geisteswissenschaftliches Asset Management System
  • GIS - Geografski informacijski sistem
  • GLAM - kratica za kulturni sektor, ki vključuje galerije, knjižnice, arhive in muzeje (Galleries, Libraries, Archives and Museums)
  • HOPE - aplikacijski profil Heritage of People's Europe
  • HTML - označevalni jezik za besedila (HyperText Markup Language)
  • INZ - Inštitut za novejšo zgodovino
  • ISO standard - standard Mednarodne organizacije za standardizacijo (International Organization for Standardization)
  • JTDH - konferenca Jezikovne tehnologije in digitalna humanistika
  • LIDO - metapodatkovni standard Lightweight Information Describing Objects
  • MARC21 standard - strojno berljivi standard za knjižnično gradivo (Machine-Readable Cataloging)
  • MG - Moderna galerija
  • MGML - Muzej in galerije mesta Ljubljane
  • MySQL - sistem za upravljanje s podatkovnimi bazami (Structured Query Language)
  • NUK - Narodna in univerzitetna knjižnica
  • OAIS - referenčni model za odprte arhivske informacijske sisteme (Reference Model for an Open Archival Information System)
  • OCLC - neprofitna organizacija na področju digitalnih knjižnic (Online Computer Library Center)
  • PARTHENOS - Pooling Activities, Resources and Tools for Heritage E-research Networking, Optimization and Synergies, evropski projekt
  • RAW file surovi zapis - surovi zapis fotografskega posnetka, digitalni negativ
  • RI INZ - Raziskovalna infrastruktura slovenskega zgodovinopisja Inštituta za novejšo zgodovino
  • SAZU - Slovenska akademija znanosti in umetnosti
  • SIP - Sprejemni informacijski paket, del OAIS (Submission Information Package)
  • SIstory - portal Zgodovine Slovenije
  • TEI - konzorcij za razvoj in vzdrževanje smernic za reprezentacijo besedil v digitalni obliki (Text Encoding Initiative)
  • TRACES - Transmitting Contentious Cultural Heritages with the Arts, evropski projekt
  • XML - razširljivi označevalni jezik (Extensible Markup Language)
  • XSLT - programski jezik za pretvorbo XML dokumentov (Extensible Stylesheet Language Transformations)
